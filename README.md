# Coffee Taste Box ☕

A simple guided coffee tasting website (client-only, no accounts, no backend).

## What it does
- Choose your tasting box (A–D or A–F)
- Pick brew method (French Press / V60 / Moka Pot)
- Brew with a beginner-friendly recipe
- Taste & track notes
- Take a quiz and get a taste profile result

## Privacy
All data stays in your browser (sessionStorage). Nothing is sent to a server.

## Pages / Hosting
- If the repo is **public**, you can enable GitHub Pages.
- If the repo is **private**, use Netlify/Vercel/Cloudflare Pages (or upgrade GitHub plan).

## Files
- `index.html` — landing
- `box.html` → `setup.html` → `brew.html` → `taste.html` → `quiz.html` → `result.html`
- `app.js` — main app logic
- `catalog.js` — shop catalog data
- `styles.css` — UI

## Local usage
Just open `index.html` in your browser.
